# ** GitHub CLI **

## bash scripts for creating, listing and removing a GitHub Repo


### Here are some bash scripts for creating a remote github repo and a related local repo with a README.md inside. 

Creation - Entering the gh usernmae, fine-grained token, reponame, specifying the privateness needed.
```bash
./createrepo.sh
```

Listing all the repos for a gh user
```bash
./listrepos.sh
```

Removing a repo. Username, reponame needed to be entered. 
```bash
./removerepo.sh
```
